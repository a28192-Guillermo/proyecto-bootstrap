# Mi Proyecto Web Estático con Bootstrap

## Descripción
Una pequeña página sobre un evento en el que hay diversos talleres de artes manuales.

## Diseño de Grid (Bootstrap)
- Header: 1 columna
- Horarios: 1 tabla de 3 columnas y 7 filas
- Formulario de inscripción: 1 columna
- Footer: 1 columna

## Componentes de Bootstrap usados
- Container (para main)
- Table (para la tabla con horarios)
- Form (para el formulario)
- Button (para enviar el formulario)
